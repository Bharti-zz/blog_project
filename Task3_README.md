# My Blog Project – Task 3

## Overview
This is **Task 3** of my Blog Project where I focused on making the blog **dynamic, searchable, and paginated**.  

Users can now:  
- View posts dynamically from the database  
- Search posts by **title** or **content**  
- Navigate posts easily using **pagination** (3 posts per page)  

All posts are displayed in **Bootstrap cards** for a clean and responsive design.  

## Features Implemented ✅

1. **Search Functionality** – Filters posts by title or content  
2. **Pagination** – 3 posts per page, works with search results  
3. **Database Integration** – MySQL `posts` table  
4. **UI with Bootstrap** – Responsive cards and buttons  
5. **Clean Data** – Duplicate posts removed  

## Tech Stack 🛠
- PHP  
- MySQL  
- Bootstrap 5  
- HTML / CSS
