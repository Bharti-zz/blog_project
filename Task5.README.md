# Blog Project + User Authentication System

This is a full-stack PHP & MySQL project demonstrating a **Blog system with search, pagination, and user authentication**.  

---

## **Features**

### **Task 1–3: Blog Project**
- Display blog posts dynamically from MySQL database
- Search posts by title or content
- Pagination to navigate through posts (3 posts per page)
- Fully responsive layout using Bootstrap

### **Task 4**
- Frontend enhancements and minor bug fixes  

### **Task 5: User Authentication System**
- User registration (signup) with username, email & hashed password
- User login and session management
- Dashboard with personalized welcome
- Logout functionality
- Dashboard link to Blog Project for easy navigation

---

## **Demo Workflow**

1. User registers via `register.php`
2. Successful login → redirect to `dashboard.php`
3. Dashboard → Go to Blog Project → view posts, search & pagination
4. Logout → redirect back to login page
